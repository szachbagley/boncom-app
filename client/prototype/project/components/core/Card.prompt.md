**Card** — flat, hairline-bordered container. No shadow, no rounding; depth comes from the border and interior whitespace.

```jsx
<Card padding="md" accent>
  <SectionLabel>Summary</SectionLabel>
  <p>…</p>
</Card>
```

Props: `padding` (none/sm/md/lg), `interactive` (hover border), `accent` (cyan top rule).
